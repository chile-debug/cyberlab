
# Cybersecurity Training Samples

This repository contains sample exercises and projects from the **Fixlab Technologies Cybersecurity Training**.  
It is intended for learning and reference purposes.

## Contents

- Network monitoring examples (Wireshark, tcpdump)  
- SIEM labs and scripts  
- IDS/IPS rules (Snort, Suricata)  
- Linux security exercises (file permissions, user management)  
- Basic digital forensics exercises
